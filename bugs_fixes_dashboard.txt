Dashboard
Sales by Category (Highest Performing Category)
	•	The Sales by Category section in the dashboard displays the highest sold category based on total sales.
	•	It also shows the total revenue generated by that category, helping users quickly identify the top-performing product category and analyze sales trends.
 
Recent Activity
	•	Displays the recently purchased items
	•	Shows the latest transactions in the dashboard
	•	Indicates whether the purchase was made using a coupon
	•	Indicates whether the purchase was made via payment gateway (direct payment)
	•	Helps track recent user purchase activity



Order Management – Invoice Download
	•	Provides an option to download the invoice for purchased coupons
	•	Allows users to access invoice details for each order
	•	Supports record keeping and transaction reference
	•	Helps users retrieve purchase invoices when needed
